# Data-analysis-Dashboard-Excel
## Project Objective
This project is an Excel-based Customer Analytics Dashboard designed to analyze customer-wise sales performance using data from two sources: Customer Information and Order Details. The dashboard provides interactive KPIs, summary insights, and customer-level trends to support business decision-making.
